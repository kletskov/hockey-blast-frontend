Project Background

This site was created while I was in-between jobs and had enough time to make it happen. It started with an idea of building a couple of plots to visualize some of the data points I was interested in here, but then it became something bigger :) I was developing on a desktop and so mobile view is not perfect on some of the pages, this should be addressed. There are multiple ideas to make it better and cooler. The list of feature requests resides in this spreadsheet.

Future of this Project

I am starting a new daytime job soon, so the features will be implemented at a much slower pace. If you have your own idea of what to add or would like to handle something from the features list, it should be straightforward. Follow the simple setup described in the frontend's README.md and then see how the code is structured. A pair of blueprint Python and template HTML files define every endpoint. Modify existing ones or create a new one, verify in your local setup that it works, and submit a request to add it to the repo.

With the help of modern LLM (I use VSCode + Copilot), there is no need to know Python, JavaScript, or HTML. Just explain what you want to do, and it will most likely do the job for you. I am looking forward to working with you and approving your requests. Let's have a BLAST!
