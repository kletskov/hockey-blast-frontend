Scorekeeper Quality Analysis - Cross-Organizational Save Button Usage Patterns

About Quality Analysis: This page identifies scorekeepers who may be "clicking the save button like crazy" by analyzing their save recording patterns across all organizations. Quality metrics help identify potential issues with frantic or excessive button usage that could indicate poor data quality or scorekeeper behavior problems.

Quality Score: Higher values indicate more concerning save button usage patterns (0-100 scale).
Peak Saves: Maximum saves recorded in 5 or 20 second windows during any single game.
Avg Max: Average of the maximum saves per time window across all games.
Rankings show worst quality first (higher percentiles = worse quality, more suspicious patterns).
Cross-Organizational: This analysis considers scorekeeper performance across all organizations they work for.
